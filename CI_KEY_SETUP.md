# 微信小程序CI私钥获取步骤

## 方法一：通过微信公众平台获取

### 1. 登录微信公众平台
访问 https://mp.weixin.qq.com/ 登录你的小程序账号

### 2. 进入开发管理
```
开发管理 → 开发设置 → 小程序代码上传密钥
```

### 3. 生成密钥
1. 点击"生成"按钮
2. 勾选"已阅读并同意《微信小程序代码上传密钥使用条款》"
3. 点击"确定"

### 4. 下载密钥文件
- 点击"下载密钥文件"
- 保存为 `private.key`（或者任何名字）

### 5. 复制密钥内容
用文本编辑器打开下载的文件，复制全部内容

### 6. 在GitHub配置
- 打开你的GitHub仓库 → Settings → Secrets and variables → Actions
- 新建Secret，名称：`PRIVATE_KEY`
- 值：粘贴刚才复制的密钥内容

---

## 方法二：如果已有密钥

如果你之前已经生成过并下载过，直接：
1. 找到之前下载的 `.key` 文件
2. 用文本编辑器打开复制内容
3. 粘贴到GitHub的 `PRIVATE_KEY` Secret中

---

## 注意事项

- **密钥有效期90天**，过期后需要重新生成
- 密钥**只能下载一次**，请妥善保存
- 如果密钥泄露，立即在公众平台重置

---

## 验证配置

配置完成后，推送代码到GitHub，在Actions中看到构建成功即可。
